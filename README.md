# PayPal Financial Forecasting

## Overview

This project forecasts PayPal's financial performance over the next twelve quarters using historical SEC filings and macroeconomic indicators.

---

## Objectives

• Forecast financial statements
• Evaluate future business performance
• Perform scenario analysis

---

## Methodology

• Holt-Winters Time Series Forecasting
• Sensitivity Analysis
• Financial Modeling

---

## Tools

• Excel
• Python
• SEC Filings

---

## Results

• 2.89% MAPE
• Data-driven financial projections
• Strategic business recommendations
